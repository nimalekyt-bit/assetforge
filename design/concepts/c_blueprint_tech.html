§ 01

Спецификация инструмента

6 узлов · допуск 0px
AF-01 · BG/REMOVEREV C

Удаление фона

Прозрачный, белый, однотонный, зелёный/синий хромакей — снимаются автоматически. Сложный фон — через ИИ (rembg). Мягкие края + despill убирает цветной ореол.

＋
AF-02 · CROP/XYREV C

Умная обрезка

Сама находит контент и разделяет несколько лого в одном файле (XY-cut). Паддинг, вписывание в квадрат или по ширине, аккуратное выравнивание.

＋
AF-03 · SCALE 16–512REV C

Нарезка размеров

Полный набор 16–512 px для иконок плюс крупные лого. Качественный ресемплинг Lanczos — чёткость даже на 16px, без каши.

＋
AF-04 · EXPORT/ZIPREV C

Форматы экспорта

PNG, ICO (многоразмерный), ICNS для macOS, WebP, SVG-обёртка. На выходе готовый ZIP и контрольный лист со всеми размерами и подписями.

＋
AF-05 · PREVIEWREV C

Живой предпросмотр

Видно результат до экспорта — на светлом, тёмном и шахматном фоне. Прозрачность, края и despill проверяются глазом, а не на угад.

＋
AF-06 · BATCH/PRESETREV C

Batch + пресеты

Обработка целых папок и готовые пресеты: favicon, Steam, Discord, Android, iOS, Windows Store. Один прогон — полный комплект под платформу.
