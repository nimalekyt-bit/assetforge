## Умные / AI-возможности обработки

| Эталон | Что предлагает | Чего нет у нас |
|---|---|---|
| **remove.bg / Photoroom** | AI-матирование волос/меха, замена фона на сцену, AI-тени, сохранение исходной тени | Только опц. rembg (груб, по умолчанию не стоит); нет замены фона, теней, качественного матирования |
| **Cloudinary g_auto** | Subject-aware кроп под любой aspect (AI-субъект + saliency + приоритет лиц) | Кроп только center/top/bottom/left/right — субъект режется под 16:9/баннер |
| **Twitter/X auto-crop** | Saliency-нейросеть предсказывает точку внимания, центрирует превью | Точка интереса не вычисляется, кропы вслепую по геометрии |
| **Real-ESRGAN / Upscayl / Topaz** | AI-апскейл 2x/4x с восстановлением деталей, офлайн (ONNX/ncnn) | Только lanczos/bicubic + unsharp — мелкое лого нельзя качественно увеличить |
| **SAM / MobileSAM** | Интерактивная сегментация по клику/рамке; MobileSAM ONNX-only, <300мс CPU | Выбор объекта только по alpha после удаления фона — нельзя кликнуть объект на сложном фото |
| **Coolors / Adobe Color** | Извлечение доминирующего цвета и палитры, экспорт hex/токенов | Нет палитры/доминирующего цвета (нужно для favicon-фона, theme_color) |
| **Lightroom / PS Neural** | Авто-уровни, авто-WB, авто-контраст, шумоподавление, Harmonize | Нет авто-коррекции тона — тусклый/жёлтый исходник уходит как есть |
| **Google Vision / docling** | Авто-классификация типа (logo/photo/icon/screenshot), EfficientNet ONNX | Нет авто-определения типа → нет авто-рекомендации пресета |
| **TinyPNG / Squoosh / Cloudinary q_auto** | Авто-качество/формат, QA-предупреждения (мелкий исходник, потеря при апскейле), AVIF | Нет QA-проверок (блюр, разрешение, шум, safe-area), нет AVIF |

## Вывод

Три повторяющихся темы, в которых конкуренты системно нас опережают:

1. **«Понимание»** — они классифицируют контент, считают saliency и предупреждают о качестве. Мы —
   геометрический ресайзер без понимания того, что обрабатываем.
2. **«Полнота доставки»** — они отдают готовые-к-вставке бандлы с манифестами и HTML-сниппетами. Мы —
   россыпь PNG.
3. **«Качество выреза и оформление»** — AI-матирование, замена фона, тени, эффекты. У нас — color-key
   и прозрачность.

Самые дешёвые победы относительно конкурентов: **favicon-пакет + HTML-сниппет** (как RealFaviconGenerator),
**AVIF + показ % экономии** (как Squoosh), **прямоугольные таргеты + saliency-кроп** (как Cloudinary).
