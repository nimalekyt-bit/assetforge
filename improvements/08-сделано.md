# 08 — Сделано (этот заход)

Реализовано и покрыто тестами (`tests/test_engine.py` — **25/25 зелёные**). Многоагентно:
ядро/интеграция — основной поток, чистые модули (`classify`/`effects`/`saliency`) — параллельные агенты.

## Tier 0 — фундамент и быстрые победы ✅
- **Прямоугольные таргеты W×H** — фундамент несквадратных ассетов. `config.NamedTarget` + `ExportConfig.targets`;
  `resize.render_target(w,h,fit,bg,focus)` с `fit ∈ {contain/pad, cover, stretch}`; `bg ∈ {None, [r,g,b(,a)], "auto"=цвет края}`.
  `build_export` рендерит `targets/<name>.png` (targets-only пресет не плодит квадратный набор). _Тест: test_rect_targets_and_jpg_retina._
- **JPG-вывод + retina @2x + непрозрачная подложка** — `format:"jpg"`, `ExportConfig.retina` (×2, суффикс `@2x`), `jpg_bg`.
- **AVIF passthrough** — `format:"avif"` через `PIL.features.check('avif')` с graceful-фолбэком на WebP + note.
- **QA-предупреждения** — `_qa_warnings`: апскейл выше исходника, контент во весь кадр без полей → `meta['warnings']`.
- **Die-cut контур** — `effects.add_outline` (адаптивная толщина), доступен через пресет `sticker`.

## Tier 1 — ядро «умности» ✅ (частично)
- **Авто-классификатор типа ассета** (`core/classify.py`): icon/logo/wordmark/photo/illustration/sprite-sheet по
  aspect, числу цветов, «плоскости», альфе, числу объектов → `suggested_preset` + `confidence` + `reasons`.
  Прокинут в `pipeline.analyze().meta`, в `/api/analyze`, и в UI инструмента (подсказка «Похоже на… → пресет», в один клик применяется). _Тест: test_classify_and_analyze_meta._
- **Saliency / subject-aware центрирование** (`core/saliency.py`): alpha + Sobel → центр значимости; авто-используется
  в `render_target(fit="cover")` (умный кроп вместо тупого центра), graceful-фолбэк к центру. _Тест: test_saliency_offcenter._
- **Эффекты подложки** (`core/effects.py`): outline / drop_shadow / rounded_corners / border / background — адаптивные,
  без обрезания (холст расширяется). Применяются в `build_export` через `ExportConfig.effects`. _Тест: test_effects_module._

## Tier 2 — расширение покрытия ✅ (контент)
- **Пресет-пак соцсетей** (`presets/social-all.json`): OG, Twitter/X, Instagram (пост+stories), YouTube, LinkedIn, Facebook —
  cover-кроп с saliency, фон «auto». _Тест: test_social_preset_targets._
- **Email/маркетинг пак** (`presets/email-marketing.json`): hero/header/signature, непрозрачный bg, PNG+JPG, retina.
- **Стикер** (`presets/sticker.json`): контур + тень.
- **Замена фона (базовая)**: `render_target(bg=...)` — сплошной/auto подложка под объект.

- **Спрайт-атлас** с JSON/CSS-картой (`core/atlas.py`, источник кадров — `detect.split_objects`).

- **Tier 3 (AI)**: матирование волос/полупрозрачности, AI-апскейл, реальная векторизация, публичный `/v1` API.

# Заход 2 — SaaS, Desktop и Админ-панель (2026-06-14)

Покрыто тестами: `tests/test_engine` (25) + `test_saas` (12) + `test_admin` (19) +
`test_hardening` (5) + `test_features` (4) = **65 зелёных**.

## Desktop-приложение (single-exe)
- **Self-installer + авто-обновление**: один `AssetForge.exe` — первый запуск ставит в
  `%LOCALAPPDATA%`, ярлыки, WebView2; дальше тихая проверка обновлений и нативное окно (pywebview).
  Флаги `--uninstall/--install-silent/--no-update`. Запасной путь — браузер.
- **Фримиум (Вариант C)**: гость (256px/PNG) → free (вход) → Pro/Business; лимиты enforce локально
  (`server/entitlement.py`), тариф из облака (`/api/desktop/auth|validate`), офлайн-грейс 7 дней.
- Брендированные окна установки/сплэша; иконка из бренд-палитры (`installer/make_assets.py`).
- Чинены реальные баги: bcrypt↔passlib, windowed-stdout (uvicorn), `webview.start(worker)`,
  повторный запуск setup = запуск приложения, отложенное удаление через .bat.

## SaaS-сайт
- Раздача десктопа: `/desktop` (страница), `/desktop/latest.json` (манифест), скачивание exe.
- Кабинет: смена имени/пароля, **удаление аккаунта**, отмена/возобновление продления,
  **2FA (TOTP)**, **API-ключи**, реферальная ссылка.
- Грейсфул-гейтинг Pro-функций в инструменте (замочки + апселл).
- Публичный **REST `/api/v1`** (`/me`, `/cutout`) с авторизацией по ключу, лимитами и квотой.
- Реферальная программа (+14 дней Pro пригласившему при первой оплате), welcome-письмо,
  напоминания об окончании подписки (`saas/tasks.py`).

## Админ-панель
- Дашборд с **Chart.js** (регистрации, выручка, тарифы, статусы платежей) + выбор периода.
- Пользователи: поиск/фильтры/пагинация, **сегменты**, **массовые действия**, карточка с
  таймлайном, выдача/снятие тарифа, сброс квоты, верификация, блок, роль, заметки, письмо,
  сброс 2FA, **force-logout**, **impersonation**, удаление.
- Платежи (фильтр, approve/reject), промокоды (CRUD + статистика), **тарифы из UI**,
  **релизы десктопа** + статистика версий, **A/B-флаги**, **когорты**, вебхук-лог,
  журнал входов, аудит, просмотр логов, здоровье системы, настройки.
- **RBAC** (support/admin/superadmin), **IP-allowlist** для админки, **IP-блок-лист** сайта,
  расписание анонса, Telegram/webhook-уведомления.

## Инфраструктура
- **Alembic**-миграции (init → admin → roles/2fa → api/webhooks/referral → login events →
  releases/flags/version-stats/session-epoch); авто-миграция колонок безопасна для Postgres.
- Опц. **Redis** (kvstore) для rate-limit и блобов сессий; диск-фолбэк; лимит сессий на пользователя.
- **CSRF** на формах, **лимит загрузки + защита от «бомб»**, единое **логирование** (+ кольцевой
  буфер для админки, опц. Sentry).
